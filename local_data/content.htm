網路連線分類 :

*NAT : Virtual server 可透過NAT取得與主電腦上網同位階,但外部連線無法

*Host-only : 若使用內建設定將取192.168.56.1群之往群IP從192.68.56.101取起windows可透過192.168.56.101對Ubuntu虛擬主機連之,但Ubuntu無法上網

*Bridged : 橋接的網卡設定可對應到實體網卡or和主電腦得網卡共連之,所以完成實體電腦和虛擬電腦位階相同則可相連線也可外連內

電腦如何開機

傳統BIOS開機流程

1.初始化

2. POST（Power On Self Test；開機自我檢測)

3. 記錄電腦系統的設定值

4. 提供常駐程式

5. 載入作業系統

 

UEFI BIOS系統的開機流程

1.SEC(Security)階段.

其主要的特色為cache as RAM，即處理器的快取當成記憶體

2. PEI（EFI前初始化）階段

用以喚醒CPU及記憶體初始化

3. DXE階段

主要功能在於溝通EFI驅動程式及硬體

4. BDS（開機設備選擇）階段

使用者就可以自開機管理者程式頁面，選擇要從哪個偵測到的開機設備來啟動

5. TSL（短暫系統載入）階段

由作業系統接手開機
